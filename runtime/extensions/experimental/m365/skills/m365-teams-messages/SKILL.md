# M365 Teams Messages

Use this skill to read messages from a Teams conversation.

## Workflow

1. Use the `m365_teams_messages` tool with the chat ID (from `m365_teams_chats`).
2. Summarize decisions, asks, owners, and deadlines.
3. Return relevant message IDs or links when available.
