# M365 SharePoint Browse

Use this skill to navigate SharePoint document libraries and OneDrive folders.

## Workflow

1. Use the `m365_spo_browse` tool with a site URL and optional folder path.
2. Return file/folder names, types, and navigable paths.
3. Use the results to drill into specific folders or identify files for download.
